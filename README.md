## ?? Getting Started  
  
### Prerequisites  
- Node.js 18.17 or higher  
- npm 9.0 or higher  
  
### Installation  
1. Clone the repository  
   git clone https://github.com/marcythany/portfolio.git  
   cd portfolio  
  
2. Install dependencies  
   npm install  
  
3. Start development server  
   npm run dev
